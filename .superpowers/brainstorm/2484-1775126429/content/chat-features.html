Interactive Chat Features for Voyager

Which features should we prioritize? Select all that excite you.

A

Clickable Option Cards

When the agent presents choices (flights, hotels, restaurants), render them as selectable cards instead of text. Click to choose, submit button to confirm. No typing needed.

B

Google Places Photo Cards

When destinations, restaurants, or experiences are mentioned, show real photos from Google Places. Image carousel or hero image per suggestion.

C

Quick Reply Chips

When the agent asks yes/no or multiple-choice questions, show clickable chips below the message. "Yes, book it" / "Show me more options" / "Change dates" — one click instead of typing.

D

Inline Budget Tracker

After each booking decision, show a mini budget bar right in the chat — visual progress of how much budget is allocated vs. remaining. Updates live as items are added.

E

Map Preview Cards

When locations are discussed (hotel near Sagrada Familia, restaurant in Gothic Quarter), show a small embedded map snippet. Static Google Maps image with a pin.

F

Rich Flight/Hotel Result Cards

Instead of text lists, show structured cards with airline logos, star ratings, price badges, and departure times. Visual hierarchy makes comparison instant.

G

Day-by-Day Timeline

When the agent builds a full itinerary, render it as a visual timeline instead of text. Each day is a collapsible card with morning/afternoon/evening slots.
